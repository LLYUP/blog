🌐 一、yum 命令简介

什么是 yum？

yum（Yellowdog Updater, Modified）是 Linux 中的包管理工具，可以用来安装、卸载、更新系统中的软件包。

常用命令：

命令 说明
yum install 软件包名 安装软件
yum remove 软件包名 卸载软件
yum update 更新所有软件
yum search 软件名 搜索软件包
